So I was reading through my blog this morning and I found a funny pattern. Anyways, here's an update to my last entry on updating my computer, it worked! I don't know how or why, it just worked. Here's a tutorial I guess for this:
1. For GPG key errors, try manually resetting keys -> removing the key folder and resetting keys
2. Try setting $DISPLAY to 0 -> unset $DISPLAY will do as well
3. For the conflicting files error try -> # pacman -Rdd linux-firmware # pacman -Syu linux-firmware
4. Also check if the time is out of sync
5. UPDATE YOUR SYSTEM EVERY 2 WEEKS!! (minimum)
Keeping your Arch install healthy is important! :)
